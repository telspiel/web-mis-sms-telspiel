SMS Portal
Overview

The SMS Portal is a React-based web application that allows clients to manage and schedule SMS campaigns with ease. It provides a user-friendly interface to create, monitor, and analyze campaign performance in real time.

Features

📅 Campaign Scheduling – Create and schedule SMS campaigns effortlessly.

📊 Reports & Analytics – View delivery reports and campaign statistics in a clear, organized format.

👥 Client Management – Manage client accounts and sender IDs efficiently.

⚙️ Responsive Dashboard – Intuitive and mobile-friendly UI built with React.

🔐 Secure Access – Role-based authentication for admins, resellers, and clients.

Tech Stack

Frontend: React.js, JavaScript (ES6+), HTML5, CSS3

State Management: Redux / Context API

APIs: RESTful API integration

Build Tool: Create React App

Getting Started

Install dependencies and run the app locally:

npm install
npm start


The app runs on http://localhost:3000
